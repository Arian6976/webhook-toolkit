# Webhook Toolkit

Simple multi-function Discord webhook tool  
(for educational & testing purposes only)

**Features:**
- Spam messages (multi-threaded, fast)
- Delete webhook
- Get webhook info (name, avatar, channel, etc.)
- Create new webhook (in your server/channel)
- Proxy support + scrape 
- Capture mode (logs successful/failed sends + responses)

# Installation
git clone https://github.com/Arian6976/webhook-toolkit.git

cd main

pip install -r requirements.txt
